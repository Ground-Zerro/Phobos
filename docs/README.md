# Документация Phobos

Phobos — комплексное решение для автоматизации развертывания WireGuard с обфускацией трафика. Включает серверные скрипты, клиентские установщики и Telegram-бот для управления.

## Основная документация

- **[README-server.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/README-server.md)** - руководство администратора VPS
- **[README-client.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/README-client.md)** - руководство пользователя клиентов (роутеры, Linux)
- **[README-bot.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/README-bot.md)** - руководство по Telegram-боту
- **[FAQ.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/FAQ.md)** - часто задаваемые вопросы
- **[TROUBLESHOOTING.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/TROUBLESHOOTING.md)** - решение проблем
- **[EXTRAS.md](https://github.com/Ground-Zerro/Phobos/blob/main/docs/EXTRAS.md)** - дополнительные сведения
